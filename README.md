# 📱 PROG7313 - Programming 3C

Welcome to **Programming 3C**! 🚀  

In this module, you will learn how to develop **native mobile applications for the Android Operating System (OS)** using the **Kotlin programming language**.

Android currently dominates the mobile market, with a **market share of about 71.69% of devices worldwide**. This means the skills you learn in this module will allow you to build applications that run on **most mobile devices in use today**.

If you previously studied **Java or C#**, you already understand many of the key **object-oriented programming (OOP)** concepts used in Kotlin. Kotlin simply introduces a **modern syntax and features** that make Android development more efficient.

Throughout this module you will create **multiple Android apps**, helping you build the practical skills needed to design, develop, and test mobile applications.

Hands-on practice is essential in programming, so make sure you complete **all activities provided on Arc**.

---

# 🎯 Module Purpose

The purpose of this module is to introduce students to **open-source software development for mobile devices**, specifically focusing on **Android application development using Kotlin**.

---

# 🧠 Module Outcomes

By the end of this module, you should be able to:

• Demonstrate knowledge of **open-source mobile software development principles**  
• Apply **development techniques used in Android app development**  
• Develop **working mobile applications to solve real-world problems**
